# Installing NimbleSM

## Configuring CMake
A very basic installation is done with:
````
cmake ${srcdir}
````
which builds and installs a default NimbleSM when you run `make`.

The full keyword listing is below.

# NimbleSM CMake Option Listing

* USE_PURE_MPI: BOOL
  * Whether to configure with MPI library.
  * Default: OFF
